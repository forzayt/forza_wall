# Contributing to fwall

We love your input! We want to make contributing to `fwall` as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features
- Becoming a maintainer


###  Submit a Pull Request
Submit a PR with a clear description of what you've changed and why.

## Code of Conduct
Please be respectful and professional in all your interactions with the project.

## Bug Reports
If you find a bug, please open an issue with as much detail as possible, including:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment (browser, OS, etc.)

## Feature Requests
We're always open to new ideas! Please open an issue to discuss your proposal before starting work on it.
